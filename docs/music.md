# 音乐播放器 (Music)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 音乐播放器**

在所有页面底部显示悬浮音乐播放器，支持网易云、QQ音乐等平台的歌单/专辑/歌手热门。

---

## 功能特性

- **黑胶唱片模式**：收起时显示旋转黑胶唱片，点击展开控制面板
- **胶囊控制面板**：展开后显示毛玻璃风格控制条，包含进度、切歌、播放列表
- **暗亮色适配**：自动跟随主题切换
- **续播记忆**：页面刷新或 PJAX 切换后自动恢复播放位置
- **PJAX 联动**：音乐播放器开启时自动启用 PJAX，确保切页不中断播放

---

## 基础设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| **启用悬浮音乐播放器** | 全局开关 | 关闭 |
| **启用 PJAX 无刷新切换** | 页面切换无刷新（音乐开启时强制启用） | 开启 |
| **音乐平台** | 网易云 / QQ / 酷狗 / 酷我 / 百度 | 网易云 |
| **资源类型** | 歌单 / 专辑 / 歌手热门 / 单曲 | 歌单 |
| **资源 ID** | 音乐平台对应资源的 ID | - |

### 获取资源 ID

**网易云音乐**：打开歌单页面，URL 中 `id=` 后面的数字即为 ID。

```
https://music.163.com/#/playlist?id=123456789
                                    ^^^^^^^^^
```

**QQ 音乐**：打开歌单页面，URL 中最后的数字即为 ID。

## 播放器配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| **主题色** | 播放器强调色 | `#fa233b` |
| **默认音量** | 0-100 | 70 |
| **播放顺序** | 列表循环 / 随机播放 | 列表循环 |
| **自动播放** | 受浏览器策略限制，大部分会被阻止 | 关闭 |
| **默认折叠歌单** | 展开面板时是否默认收起歌单列表 | 开启 |
| **歌单最大高度** | 歌单列表最大高度（px） | 250 |

## CDN / 代理设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| **APlayer CSS** | APlayer 样式文件地址 | jsDelivr CDN |
| **APlayer JS** | APlayer 脚本文件地址 | jsDelivr CDN |
| **MetingJS** | MetingJS 脚本地址 | jsDelivr CDN |
| **Meting API 代理** | 歌曲数据接口地址 | `api.injahow.cn` |

> **重要**：Meting 官方 API 已失效，必须配置可用的第三方代理。地址格式中 `:server`、`:type`、`:id`、`:r` 为占位符，会自动替换。

### 自建 Meting API

如需自建代理，可参考：

- [Meting-API](https://github.com/injahow/meting-api) — PHP 版
- 部署后将地址填入，格式示例：`https://your-api.com/meting/?server=:server&type=:type&id=:id&r=:r`

## PJAX 说明

| 场景 | PJAX 状态 |
|------|-----------|
| 音乐播放器 **开启** | 强制启用（切页不中断播放） |
| 音乐播放器 **关闭** + PJAX 开关 **开** | 启用 |
| 音乐播放器 **关闭** + PJAX 开关 **关** | 关闭，传统全页刷新 |

---

## 注意事项

1. **浏览器自动播放限制**：首次访问时浏览器会阻止自动播放，需用户点击页面后才能播放
2. **CDN 可用性**：如 jsDelivr 不可用，请替换为其他 CDN
3. **Meting API**：默认 API 可能不稳定，建议自建或使用可靠的第三方代理
4. **移动端**：iOS Safari 对后台音频有额外限制
