# 友情链接设置 (Links)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 友链**

需安装 `plugin-links` 插件。

---

## 页面设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **页面标题** | 友情链接页面的标题 | "友情链接" |
| **页面描述** | 友情链接页面的描述文字 | "感谢这些优秀的网站与博客，让互联网变得更加精彩" |
| **申请区标题** | 友链申请区的标题 | "想要交换友链？" |
| **申请区描述** | 友链申请区的说明文字 | "如果你也有一个优质的网站或博客，欢迎与我交换友链..." |

---

## 本站信息设置

配置点击“本站信息”弹窗中展示的内容，方便访客复制友链参数。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **Logo 地址** | 站点的 Logo 图片链接 | 系统默认 Logo |
| **站点名称** | 用于互换的站点名称 | 系统默认标题 |
| **站点描述** | 用于互换的站点描述 | 系统默认副标题/SEO描述 |
| **RSS 地址** | 站点的 RSS 订阅链接 | `/rss.xml` |
| **联系邮箱** | 站长的联系邮箱（显示在弹窗中） | 留空不显示 |

---

## 侧边栏设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **显示侧边栏** | 启用后显示侧边栏 | 关闭 |
| **定制侧边栏** | 启用后可单独配置 | 关闭 |
| **侧边栏位置** | 左侧 / 右侧 | 右侧 |
| **小工具列表** | 拖拽排序添加小工具 | - |

### 可用小工具
欢迎卡片（天气）、作者卡片、博客统计、文章分类、标签云、热门文章、最新文章

---

## 交互功能

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **启用评论** | 在友链页面底部显示评论区 | 开启 |

---

## 悬浮控制栏

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **定制悬浮控制栏** | 启用后可单独配置 | 关闭 |
| **Dock 菜单** | 选择菜单 | primary |
| **返回顶部按钮** | 是否显示 | 开启 |
